# K3Supervisor

Simple K3S' control panel with Laravel, using its ReST API

UI Based on my previous project, [MikroKontrol](https://https://github.com/freemann350/MikroKontrol).

## Features:

- List Nodes

- Namespaces

  - List/Create/Delete

- Pods

  - List/Create/Delete

- Deployments

  - List/Create/Delete

- Services/Ingress

  - List/Create/Delete
