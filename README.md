# K8Supervisor

Simple K8S control panel with Laravel, using its ReST API

UI Based on my previous project, [MikroKontrol](https://https://github.com/freemann350/MikroKontrol).

## Features:

- List Nodes
- Namespaces

  - List/Create/Delete
- Pods

  - List/Create/Delete
- Deployments

  - List/Create/Delete
- Services

  - List/Create/Delete
- Ingress

  - List/Create/Delete
- Platform roles

  - Verbs
    - get
    - create
    - delete
  - Resources
    - nodes
    - namespaces
    - pods
    - deployments
    - services
    - ingresses
